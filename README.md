# Card Madness Community Packs

Discover and play "high-quality" pre-made card packs for [Card Madness](https://github.com/chielre/card-madness).  All packs listed in this repository are made compatible with the game and are released under an open-source license. For installation and usage instructions, please refer to the [game repository](https://github.com/chielre/card-madness), where we explain how to integrate and use this repository within your game.
